# Contributing

## Setup

```bash
git clone https://github.com/HH-v/markdown-toc
cd markdown-toc
pip install -e .
```

## Running tests

```bash
pip install pytest
python -m pytest tests/ -v
```

## Adding features

1. Fork the repo
2. Create a feature branch
3. Add tests for new functionality
4. Submit a pull request

## Code style

Follow PEP 8. Use type hints where possible.
